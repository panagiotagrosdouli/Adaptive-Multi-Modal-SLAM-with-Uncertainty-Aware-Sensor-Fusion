# Website

Next.js + TailwindCSS + Framer Motion scaffold for a robotics lab-style project page.

Required sections:

- hero section with the research question;
- system architecture;
- sensor fusion animation;
- generated demo GIF;
- mathematical intuition;
- uncertainty visualization;
- experiments and benchmark status;
- roadmap;
- GitHub link;
- citation.

The website should only display real benchmark numbers when the corresponding metrics and reproduction manifests are committed under `results/`.
